

### 交叉工具链说明

SDK需要用到2个交叉工具链，如下：

| 目录(prebuilts/gcc/linux-x86/arm)                   | 说明                       |
| :----                                               | ------                     |
| gcc-arm-8.3-2019.03-x86_64-arm-linux-gnueabihf      | 用于文件系统和上层应用编译 |
| gcc-linaro-6.3.1-2017.05-x86_64_arm-linux-gnueabihf | 用于U-Boot和Linux内核编译  |
